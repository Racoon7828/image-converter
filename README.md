# 이미지 변환기

AI 배경제거 기능이 내장된 로컬 이미지 변환 도구입니다.  
외부 API 없이 PC에서 직접 실행되며, 사용량 제한·요금 없음.

---

## 실행 방법

```bash
pip install -r requirements.txt
python app.py
```

---

## 지원 포맷

| 입력 | 출력 |
|------|------|
| PNG, JPEG, WebP, BMP, TIFF, GIF, SVG | PNG, JPEG, WebP, BMP, TIFF, SVG |

---

## 주요 기능

- **포맷 변환** — 지원 포맷 간 자유 변환
- **크기 조절** — 너비/높이 지정, 비율 유지 옵션
- **품질 조절** — JPEG·WebP 출력 품질 슬라이더 (1–100)
- **AI 배경제거** — 로컬 딥러닝 모델로 배경 자동 제거
- **미리보기** — 파일 이름 클릭 시 메인 화면에서 즉시 확인
- **배치 처리** — 여러 파일·폴더 한 번에 변환

## 버전 히스토리

### v1.0 — 초기 릴리즈

기본 이미지 변환 기능 구현.

- PNG / JPEG / WebP / BMP / TIFF / SVG 포맷 변환
- 크기 조절 (비율 유지 옵션 포함)
- JPEG·WebP 품질 슬라이더
- AI 배경제거 체크박스 (u2net 모델)
- 파일·폴더 일괄 추가, 진행률 표시
- 2열 레이아웃 (파일 목록 | 변환 설정)

---

### v1.1 — 배경제거 품질 개선

배경제거 정확도를 대폭 향상.

**변경 사항 (`converter.py`)**

| 항목 | v1.0 | v1.1 |
|------|------|------|
| AI 모델 | u2net | **BiRefNet-general** |
| 엣지 처리 | 없음 | **Alpha Matting 적용** |
| 세션 관리 | 매번 재로드 | **모듈 레벨 캐싱** |

- `get_rembg_session()` 도입 — 앱 실행 중 모델을 한 번만 로드해 재사용
- Alpha Matting 파라미터: foreground threshold 240 / background threshold 10 / erode size 10
- `rembg` 최소 버전 `>=2.0.57` 로 상향 (BiRefNet 지원 버전)

**변경 사항 (`app.py`)**

- 프리로드 함수(`_preload_rembg_model`)가 `get_rembg_session()`을 직접 호출하도록 수정 → 모델 이중 로드 방지

---

### v1.2 — BiRefNet-massive 모델 업그레이드

더 큰 모델로 교체해 세밀한 경계(머리카락, 털 등) 처리 품질 향상.

**변경 사항 (`converter.py`)**

| 항목 | v1.1 | v1.2 |
|------|------|------|
| AI 모델 | birefnet-general | **birefnet-massive** |
| 모델 크기 | ~500 MB | ~1 GB |

- 코드 변경은 `new_session()` 인수 한 줄
- 토큰·계정 불필요, 첫 실행 시 자동 다운로드 후 로컬 캐시

---

### v1.3 — UI 3열 레이아웃 & 인라인 미리보기

메인 화면 구조를 전면 재편.

**레이아웃 변경**

```
v1.0–1.2                      v1.3
┌──────────────┬──────────┐   ┌──────────┬────────────┬──────────┐
│  파일 목록   │ 변환 설정 │   │ 파일 목록 │  미리보기  │ 변환 설정 │
│  (좌, 60%)  │ (우, 40%) │   │ (좌, 1)  │ (중앙, 3)  │ (우, 1)  │
└──────────────┴──────────┘   └──────────┴────────────┴──────────┘
```

**변경 사항 (`app.py`)**

- 창 크기 860×600 → **1100×620**, 최소 크기 720×480 → 900×500
- `grid_columnconfigure` 2열 → 3열 (weight 1:3:1)
- `_build_preview()` 메서드 추가 — 중앙 미리보기 패널
- `_show_preview()` 팝업 방식 → **인라인 패널 업데이트 방식**으로 변경
- 파일 이름 라벨에 `cursor="hand2"` 적용 (클릭 가능 표시)
- SVG는 미리보기 패널에 안내 텍스트 표시

## AI 배경제거 상세

| 항목 | 내용 |
|------|------|
| 모델 | BiRefNet-massive |

| JPEG·BMP 출력 시 | 배경이 흰색으로 채워짐 (투명도 미지원 포맷) |
